🐍 Developer Setup (Optional)

If you're building from source or fixing missing DLLs:

  1. Install Python 3.12 (32 or 64-bit based on your OS)
  2. Create a virtual environment: 
    python -m venv redai_env
  3. Activate it: 
    redai_env\Scripts\activate
  4. Install required modules: 
    pip install -r requirements.txt
  5. Run: 
    python redai_main.py --plugin network_scan
